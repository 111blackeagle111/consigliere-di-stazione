# consigliere-di-stazione
Consigliere di Stazione - Sistema di logging e analisi predittiva per stazioni radioamatoriali. Stack: Python/FastAPI, SQLite, SQLAlchemy, integrazione Ollama (Llama 3.2) per AI locale. Monitoraggio propagazione via API NOAA. Frontend responsive vanilla HTML/JS. Architettura zero-cloud, privacy-first.
